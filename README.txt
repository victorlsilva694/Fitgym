Com muito esforço, tenho mais um projeto finalizado... Dessa vez um sistema de condicionamento físico e saúde chamado FitGym. 

Tutorial para instalar e utilizar o projeto: 

Primeiro, abra um terminal linux ou CMD no windows
 
Em seguida digite git clone https://github.com/victorlsilva694/Fitgym.git para que possa baixar o repositório

Para conseguir rodar o sistema digite para Linux o comando: npm run start

Ou se caso usar o Windows digite: npm start

O sistema vai rodar no http://localhost:3000 se ocorrer tudo normalmente

É necessário ter na máquina o mysql.
